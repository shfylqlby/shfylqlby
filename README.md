## Tentang Saya

Halo! Saya **Qalby** — Network Engineer dari **Pekalongan, Indonesia** 🇮🇩

-  Spesialis **MikroTik RouterOS** & manajemen **WAN** skala ISP
-  Aktif membangun sistem **otomatisasi jaringan** via scripting & **WhatsApp Gateway API**
-  Mengelola infrastruktur **Proxmox VE**, Linux server, dan layanan web production
-  Berpengalaman setup **RT/RW Net**, jaringan kantor, hingga ISP mikro skala lokal
